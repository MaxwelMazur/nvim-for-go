# config-nvim
<p>My configuration nvim for all type in work</p>

> install neovim in fedora
```
sudo dnf install neovim 
```
<br>

> install config 
```
cd ~/.config/ && git clone https://github.com/MR3MMZ/nvim.git
```
<br>

> open neovim
```
nvim
```
